## Como rodar o código

Compilar:

`ghc -o Main Main.hs`

Executar:

`./Main`

----

## Como usar o git / github

- Atualizar o repositório:
`git pull`

- Verificar quais arquivos foram modificados:
`git status`

- Verificar quais mudanças foram feitas:
`git diff` ou no terceiro icone do VS Code

- Criar uma branch de trabalho:
`git branch <nome>`

- Trocar de branch:
`git checkout <nome>`
